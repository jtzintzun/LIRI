# LIRI
Language Interpretation and Recognition Interface

LIRI is an awesome and powerful node.js application that grants you access to handy information from Tweeter, Spotify, and IMDB. 

Because it is better to get information that gets data. LIRI helps you to get quick access to the relevant information you want based on the parameters you to the app. 

Using NPM packages; LIRI gets access to data from Tweeter, Spotify, and IMDB. That allows the users to modify the search criteria and the information displayed, just rewriting a few lines in the code.
